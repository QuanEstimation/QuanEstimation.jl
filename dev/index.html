QuanEstimation.jl

QuanEstimation.jl is an open-source Julia framework for scheme evaluation and design in quantum parameter estimation. It can be used either as an independent package or as the computational core of the Python-Julia package QuanEstimation.

Key Features

  • Comprehensive toolkit for quantum parameter estimation

Installation

To install QuanEstimation.jl:

using Pkg
Pkg.add("QuanEstimation")

Quick Start

using QuanEstimation
