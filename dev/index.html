QuanEstimation.jl

QuanEstimation.jl is an open-source Julia framework for scheme evaluation and design in quantum parameter estimation. It can be used either as an independent package or as the computational core of the Python-Julia package.

Key Features

  • Dynamics evolution (matrix exponential / ODE) with derivatives
  • Kraus operator parameterization
  • Quantum and classical Fisher information (QFI, QFIM, CFI, CFIM)
  • Asymptotic bounds: SLD, RLD, LLD, HCRB, NHB
  • Bayesian bounds: BCRB, BQCRB, VTB, QVTB, QZZB, OBB
  • Bayesian estimation and cost bounds
  • Control, state, measurement, and comprehensive optimization
  • Adaptive measurement schemes
  • Metrological resources: spin squeezing, minimum-time search

Installation

using Pkg
Pkg.add("QuanEstimation")

Quick Start

using QuanEstimation

# Initial state
rho0 = 0.5 * ones(2, 2)

# Free Hamiltonian and derivatives
sz = [1. 0.0im; 0. -1.]
H0 = 0.5 * sz
dH = [0.5 * sz]

# Dynamics
tspan = range(0.0, 10.0, length=2500)
rho, drho = QuanEstimation.expm(tspan, rho0, H0, dH)

# Quantum Fisher information
F = QuanEstimation.QFIM(rho[end], drho[end])

  • Parameterization — define system dynamics (Hamiltonian, decay, Kraus operators)
  • Metrological Tools — Fisher information, Cramér-Rao bounds, Bayesian estimation
  • Control Optimization — GRAPE, PSO, DE algorithms
  • State Optimization — AD, PSO, DE, Nelder-Mead, RI
  • Measurement Optimization — projection, linear combination, rotation
  • Comprehensive Optimization — joint optimization of state, control, and measurement
  • Adaptive Schemes — online/offline adaptive measurement
  • Resources — spin squeezing, minimum time
  • Output Files — file formats and post-processing
